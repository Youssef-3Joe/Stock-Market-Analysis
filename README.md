📈 Alpha-Stream: Automated Finance Intelligence

"What is this?" 🤔
I built a system that doesn't just show stock prices—it thinks about them. This project is a full-scale data pipeline that watches the market, handles the messy math, and tells you where the "Safe Havens" are versus the "Risk Traps."

🛠️ How It Works (The "Boring" But Cool Part)
📥 Smart Extraction: A Python engine that only pulls new data every day (no wasting time on old data!).

🧹 The Clean-Up: Custom logic that fixes market holidays, timezone gaps, and messy duplicates.

🤖 Total Automation: Runs itself every night at 11:00 PM Cairo Time using GitHub Actions. I sleep, the data updates. 😴✨

🧠 Math Engine: Uses OLS Regression and Advanced Statistics to find the truth behind the price.

🎯 The Expert Verdict (The Results)
After crunching the numbers, here is what the data actually tells us:

🏆 Gold is the Efficiency King: With a Sharpe Ratio of 1.18, Gold was the "smartest" investment. It gave the best profit for the least amount of stress.

🍎 The Apple Amplifier: Apple has a Beta of 1.20. This means it’s a "Market Mirror" on steroids—it wins bigger when the market is up, but crashes harder when it's down.

⚡ The Tesla Rollercoaster: My Kurtosis analysis shows Tesla's wild moves are "normal" for it. It’s high-risk, high-reward, and not for the faint of heart!

💻 Tech Stack
Language: Python 3.10 🐍

Data: Pandas, NumPy, YFinance, matplotlib, seaborn, plotly, datetime, os, sqlalchemy, urllib 📊

Statistics: Scipy, Statsmodels, OLS Regression 📉

Automation: GitHub Actions (CI/CD) ⚙️

UI: Streamlit 🚀
